# Github Search

> Simple Github Search App

[Live Demo](https://epilot-assignment.vercel.app/)

## Technologies Used

- Vite + React + Typescript
- React Router@6
- TailwindCSS
- Playwright for e2e testing

## How to run locally

Install dependencies

```sh
npm install
```

_Development_

```sh
npm run dev
```

_Production build_

```sh
npm run build
npm run preview
```

## e2e Testing

Used playwright with UI mode for running the e2e tests.

```sh
npm run test:e2e
```
